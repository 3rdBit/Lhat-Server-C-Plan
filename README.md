## 教程 TUTORIAL
### 配置设置文件 CONFIGURATION
配置文件是一个JSON文件，里面有一些配置信息，比如端口、是否记录日志等等。  
请在server.json里面配置好服务器的属性。
### 启动服务器 LAUNCHING
在终端输入：
`server.exe`  
即可启动服务器。
